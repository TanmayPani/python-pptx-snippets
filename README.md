## pptx-python-snippets


Extra snippets for some more functionalities with `python-pptx`. 

Currently they revolve around adding more functionalities to movie shapes, but will keep adding more as I use the package more...


## How to run
Assuming you're already in an environment with the dependencies installed (mainly `pptx-python` and whatever it comes with.)

- Clone this repository or just take the `snippets.py` file. 

### Testing
- If you wish to test it before starting, just do the `git clone ...` or download atleast `snippets.py`, `example_vide0.mp4` and `main.py` 
- `main.py` is a marimo notebook showing detailed usage and explanations which can be viewed [here](https://molab.marimo.io/github/TanmayPani/python-pptx-snippets/blob/main/main.py) or opened using `marimo edit`
- Run `python main.py` or `uv run main.py` if using `uv` (highly recommended) and open the output `output.pptx` in PowerPoint or Keynote

## Caveats 
- So far can't guarantee these working with LibreOffice Impress or the other open-source PowerPoint alternatives. If using linux then I personally recommend using markdown or `beamer`(`\Latex`) to make slides programatically.
